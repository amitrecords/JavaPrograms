This project is my own implementation of the Unistroke recognizer (1$ Recognizer) in Java created with the help of single training example corresponding to each of the 16 possible classes of gestures listed as follows:
1. Triangle
2. Arrow
3. Circle
4. Rectangle
5. Check
6. X
7. Delete
8. Caret
9. V
10. Left square bracket
11 Right square bracket
12. Left curly bracket
13. Right curly bracket
14 Pigtail
15 Star
